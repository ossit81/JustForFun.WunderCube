

This is a solution for Wundercube game (http://www.wunderdog.fi/wunderkuutio). 

This application needs .Net Framework 4.5.2 or higher.

Start application from command line with following command:

C:\<your application folder>\JustForFun.WunderCube.exe

Application will read from execution folder 'cube.txt' and 'words.txt' files. The files can be found from Wunderdog website: http://www.wunderdog.fi/s/cube.txt, http://www.wunderdog.fi/s/words.txt or from solution folder ..\JustForFun.WunderCube\Data\. 

After the application has been executed successfully, the list of resolved words can be found from output file 'results.txt'.

Enjoy.
